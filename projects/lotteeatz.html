롯데이츠 전자영수증

롯데GRS에서 진행하는 롯데잇츠 프로젝트에 참여하여 전자영수증 Rest API를 개발하였습니다.

개발단계는 사내에서 진행하였고 배포는 롯데GRS 개발 담당자에게 WAR파일을 전달하면 담당자가 JEUS에 배포하는 방식으로 진행하였습니다.

참여인원 및 업무
  • 팀장 - 업무 협의
  • 웹개발자 1명 - 영수증 화면 Rest API 개발
  • 퍼블리셔 1명 - 웹 퍼블리싱(HTML, CSS, IMAGE)
My 상세 업무
  • POS(오프라인)결제, 온라인결제, 스마트영수증에 대한 POS 데이터를 조회하여 HTML형태로 파싱하여 응답하는 Rest API를 개발.

애플리케이션 구성

  • Oracle Java 1.8
  • Springframework 4.3.25.RELEASE
  • JNDI를 사용하여 Jeus에 등록된 DataSource(MS-SQL) 사용
  • Mybatis
  • Maven
  • lombok

Rest API

제공한 Rest API로 요청이 오면 데이터베이스에서 해당 결제정보를 조회하여 영수증화면 HTML로 변환하여 응답을 줍니다.
