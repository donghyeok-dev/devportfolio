과학기술인공제회 CMS(Cash Management Service)

과학기술인공제회와 금융결제원의 CMS 연계 시스템을 개발하는 프로젝트입니다.

개발단계는 사내에서 진행하였고 배포 및 테스트 단계는 과학기술인공제회 본사에 며칠간 상주하면서 진행하였습니다.
사내에서는 금융결제원과의 테스트가 불가하여 금융결제원에서 제공받은 개발 가이드 문서만 가지고 사내에 테스트 송,수신 서버를 만들어서 개발 및 테스트를 진행하였습니다.

참여인원 및 업무
  • 팀장 - 업무 협의
  • 개발자 1명 - 시스템 개발
  • 퍼블리싱은 과학기술인공제회에서 제공

My 상세 업무
  • CMS파일 송수신 모듈 개발 - 출금이체신청결과내역(EB12), 출금동의증빙자료(EI13), 출금이체신청내역(EB13), 출금이체(익일출금)의뢰내역(EB21), 출금이체(당일출금)의뢰내역(EC21), 입금이체의뢰내역(EB31)
  • CMS 송수신 모듈 감시 및 재시작 프로세스 개발
  • 관리자 웹 개발 - CMS 파일 별 송신 시간 설정, 프로세스 실행상태 및 재시작, 송수신 이력 및 결과 확인

애플리케이션 구성

  • CMS송수신모듈 - Oracle Java 1.6
  • 프로세스 감시 모듈 - Oracle Java 1.6, Netty
  • 웹 관리자 - Oracle Java 1.6, Springframework 3, Mybatis, Maven, javascript, Jquery, socket.io

서버 구성

CentOS 서버 1대, 데이터베이스(Oracle) 서버, 금융결제원 서버로 구성되어 있으며,
CentOS 서버는 CMS 송수신 모듈, Tomcat(웹 관리자), 프로세스 감시 모듈로 구성되어 있습니다.

CMS송수신모듈

과학기술인공제회 데이터베이스에 등록된 데이터와 첨부파일을 전문형식에 맞는 byte로 가공하여 금융결제원과 TCP통신으로 송수신합니다.



송수신은 금융결제원의 송수신 처리표를 참고합니다.



모듈 디렉토리는 다음과 같이 구성됩니다.



웹 관리자

갤러리아 전자영수증 발급 조회 및 통계, 마케팅 관리 등을 관리자가 이용할 수 있도록 웹으로 개발된 백오피스입니다.
