파라다이스 호텔 전자영수증

파라다이스 호텔에서 발급하는 전자영수증을 개발하는 프로젝트입니다.

개발단계는 사내에서 진행하였고 배포 및 테스트 단계는 VPN 접속을 통해 진행하였습니다. 개발팀은

전자영수증에서 웹 개발 항목은 아래와 같습니다.
- 배치를 통해 고객 휴대폰으로 전자영수증 URL 문자로 전송.
- 전자영수증 리스트 화면.
- 전자영수증 발급 현황 달력 화면.
- 전자영수증 상세화면.
- 상세화면 PDF 다운로드 기능.

서버 구성 및 배포

Cent OS WEB/WAS 서버(JEUS, Java8) 1대, 운영 데이터베이스(Oracle) 서버로 구성되어 있으며,
배포는 VPN망을 통해 SFTP로 서버에 WAR 파일을 업데이트한 뒤 Tomcat을 재기동 하였습니다.

애플리케이션 구성

- Oracle Java 1.7
- Springframework 3.2.1.RELEASE
- JDBC
- Mybatis
- Maven
- front단은 JSP 파일에 퍼블리싱된 css, js, html을 적용하고, javascript와 Jquery를 이용하여 작업.

인터페이스

데이터베이스에 문자 발송 정보를 INSERT하면 발송되는 형식.
