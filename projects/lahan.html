라한호텔 멤버십 백엔드 및 백오피스 개발

라한호텔의 멤버십 백엔드 API와 백오피스를 개발 하였습니다.
API 개발은 홈페이지 프론트 개발 업체(블루웨이브)와 호텔 시스템(야놀자, (구)산하정보통신)과 협업하여 진행하였습니다.

참여인원 및 업무
  • 개발자 1명 - 업무 협의, 설계, 개발
My 상세 업무
  • 업무,일정 협의 및 DB 설계
  • 라한 멤버십 테스트 서버 구축 (Oracle, Nginx, Java 등)
  • 회원 API 개발 - 회원가입, 아이디/비밀번호 찾기, 로그인, 마이페이지, 내정보수정 등
  • 쿠폰 API 개발 - 쿠폰 발급(회원 가입 시), 마이쿠폰, 쿠폰 조회/예약/사용/취소 등
  • 포인트 API 개발- 마이포인트, 포인트 사용/취소 등
  • 백오피스 개발 - 회원, 포인트, 쿠폰, 호텔시스템 인터페이스, 호텔 별 포인트 정산 등

  • Oracle 19c

  • Spring Boot 2.6.6
