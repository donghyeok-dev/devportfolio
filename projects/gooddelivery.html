이전 페이지로

서로조은배달 백엔드 및 백오피스 개발

서로조은배달 앱의 백엔드 API, 매장 프로그램 백엔드 API, 관리자용 백오피스를 개발 하였습니다.

참여인원 및 업무
  • 백엔드 개발자 1명
  • 안드로이드 개발자 2명
  • 아이폰 개발자 1명
  • 디자인 & 퍼블리셔 1명
My 상세 업무
  • Spring Cloud Gateway, Eureka, Config, Kafka를 활용한 이벤트 기반 미니 서비스 아키텍처 구성
  • 앱 주문 결제, 리뷰, 매장, 회원, 포인트 RESTful API 개발
  • Elasticsearch Cluster 구축 및 적용하여 매장 및 메뉴 검색 성능을 향상
  • Redis Cache를 적용하여 자주 검색되는 매장, 회원 상세 정보 검색 성능을 향상
  • Kafka Cluster 구축하여 도메인간 이벤트 및 트랜잭션 처리
  • Google Firebase 연동하여 주문 상태 알림 푸시 전송
  • 매장 POS 로그인, 주문, 리뷰 등 RESTful API 개발
  • vue.js와 vuetify로 백오피스 개발
  • Spring batch로 매장 정산 데이터 생성
  • Prometheus와 Grafana를 이용해 서버와 서비스를 모니터링
  • Sentry로 Service Error Log 수집 및 Slack Alert
  • Spring Rest Docs로 API 문서화
